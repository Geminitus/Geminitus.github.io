Personal Projects

eTutor

A social networking site with the goal of bringing users together to practice their skills in foreign languages by speaking with other users that natively speak the language. This project was primarily made using the Python framework, Django, but also uses Node.js to implement Voice and Video calling as well as instant messaging.

Languages used: HTML5, CSS, Javascript, and Python

eTutor Github

Habit Tracker

A simple Webapp that lets users create and schedule various activities that they wish to complete. This app also allows a user to let others see the progress that they make on each task as well as commenting on the tasks themselves.

Languages used: HTML5, CSS, Javascript, and Python

Habit Tracker Github

ChunkCascade

This project was made with the goal of allowing users visiting the site to ask questions to other users that may have more knowledge on the topic they are asking about, similar to Yahoo Answers or Stack Overflow. ChunkCascade uses Django for storing users data and Javascript to construct the user facing part of the project.

Languages used: HTML5, CSS, Javascript, and Python
